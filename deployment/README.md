# Развёртывание onto-emitter-gpl

Этот пакет предоставляет инструменты для развёртывания **onto-emitter-gpl** в изолированной, лицензионно-соответствующей и этически контролируемой среде.

## Требования
- Docker 24.0+ или Podman (совместимый с Docker API)
- Kubernetes 1.25+ (опционально)
- Python 3.10+ (для скриптов развёртывания)

## Режимы развёртывания
1. **Локальный (Docker)** — для тестирования и инкубации.
2. **Kubernetes** — для продакшена с поддержкой VMA-аудита и высоконадёжных сценариев.

## Важно
- Все развёртывания **должны** использовать только материалы из `data/incubation/` или явно одобренные `onto144` профили.
- Использование внешних данных без VMA-валидации **запрещено** лицензией GPL-3.0 в сочетании с ethical_rules.yaml.
- Перед запуском убедитесь, что `LICENSE`, `SECURITY.md` и `ethical_rules.yaml` доступны в контейнере.

См. `.env.example`, `secret.example.yaml` и `configmap.yaml` для настройки.
